Cambiar Contraseña

Por razones de seguridad, se requiere que el usuario cambie su contraseña de inicio de sesión la primera vez que accede al sistema. Una vez modificada, deberá iniciar sesión nuevamente.

Considere esto para una contraseña fuerte:

  • • Al menos una minúscula.
  • • Al menos una mayúscula.
  • • Al menos un número.
  • • Mínimo 8 caracteres.
